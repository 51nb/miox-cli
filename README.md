# Installation

```bash
npm install -g miox-cli
```

# Usage

**How to create a project?**

```bash
miox create
```

**How to check version?**

Check miox-cli version:

```bash
miox version
```

Check project versions(local):

```bash
miox version -p
```

Check project versions(remote):

```bash
miox version -p -l
```

Check module version:

```bash
miox version miox-router -l
```

**help?**

```bash
miox -h
```
